📋 Правила игры

Сценарий: Финансовая компания с 3 счетами в кризисе. Два менеджера по очереди переводят средства в блокчейн. Менеджер, который не может сделать ход (все счета обнулены), — уволен.

Правила: В свой ход выбери любой счёт с монетами и переведи любое ненулевое количество. Тот, кто переводит последние монеты — побеждает, другой уволен!

Стратегия (XOR-нейм): XOR всех трёх счетов = ваш «индикатор позиции».
● XOR ≠ 0 — вы в выигрышной позиции (можно навязать XOR=0 сопернику).
● XOR = 0 — вы в проигрышной позиции при идеальной игре соперника.
